# 全功能数字闹钟

一款基于 MAX II EPM240T100C5 开发板设计的简单数字闹钟。

# 介绍
## 开发环境
本项目在 Quartus II 5.0 环境下开发，使用的闪存可编程逻辑器件为 MAX II EPM240T100C5。

支持以下特性:
- 精确到秒的时间显示
- 支持调校时间
- 支持设置闹钟
